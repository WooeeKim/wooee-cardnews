# 번역투·AI티 렌즈 (UI 카피용)

한국어 UI 카피 어색함의 8할은 **영어 UI를 직역한 흔적**이다. 영어 string을 그대로 옮기면 문법은 맞아도 한국 사람은 그렇게 말하지 않는다. 아래는 인터페이스 문구에 자주 나오는 패턴과 치환.

> 이 렌즈는 MIT 라이선스 [epoko77-ai/im-not-ai](https://github.com/epoko77-ai/im-not-ai)의 한국어 AI 티 분류 작업에서 영감을 받아, UI 마이크로카피에 자주 나오는 항목만 추려 재구성했다.

## 1. success 직역 — "성공적으로 ~되었습니다"

영어 "Successfully saved"의 직역. 한국어에선 결과만 말하면 된다.
- "성공적으로 저장되었습니다" → **"저장했어요"**
- "성공적으로 전송되었습니다" → **"보냈어요"** / "메일로 보냈어요"
- "결제가 성공적으로 완료되었습니다" → **"결제를 마쳤어요"** (금지어 규칙 있으면 그에 맞게)

## 2. 막연한 오류 통보 — "오류가 발생했습니다"

"An error occurred"의 직역. 원인도 해법도 없어 사용자를 막다른 길에 둔다.
- "오류가 발생했습니다" → **"연결이 끊겼어요. 다시 시도해 주세요."** (원인+행동)
- "알 수 없는 오류" → 최소한 다음 행동이라도: **"문제가 생겼어요. 잠시 후 다시 시도해 주세요."**

## 3. 가능 표현 남발 — "~하실 수 있습니다"

"You can ~"의 직역. 안내문마다 붙어 늘어진다. 능동 청유로 바꾼다.
- "지금 시작하실 수 있습니다" → **"지금 시작해 보세요"** / "지금 시작하기"
- "여기에서 확인하실 수 있습니다" → **"여기에서 확인해요"** / "확인하기"

## 4. 피동 남발 — "~되어집니다 / ~되었습니다"

영어 수동태의 직역. 한국어는 행위자 시점 능동이 자연스럽다.
- "파일이 업로드되었습니다" → **"파일을 올렸어요"**
- "변경 사항이 저장되어집니다" → **"바뀐 내용을 저장해요"** ("되어진다"는 이중피동, 비문)

## 5. 책임 회피·사용자 탓 — "잘못 입력하셨습니다"

영어 "Invalid input"이 한국어 존댓말과 만나면 사용자를 탓하는 말투가 된다.
- "잘못 입력하셨습니다" → **"이메일을 다시 확인해 주세요"**
- "유효하지 않은 비밀번호입니다" → **"비밀번호는 8자 이상이어야 해요"** (무엇을 고칠지)

## 6. 번역체 호칭 — "귀하 / 본인 / 사용자님"

- "귀하의 계정" → **"내 계정"** / 호칭 생략 ("계정")
- "본인 확인이 필요합니다" → **"본인 확인을 해주세요"** 정도로, 혹은 맥락상 "한 번 더 확인할게요"

## 7. 명사 나열·딱딱한 한자어 통보

영어 라벨을 명사로만 옮긴 경우. 동사로 풀면 사람 말이 된다.
- "결제 진행" → **"결제하기"** (또는 금지어 규칙대로)
- "정보 수정 완료" → **"정보를 수정했어요"**
- "작업 처리 중" → **"처리하고 있어요"** / "잠시만요"

## 8. 군더더기 연결어 — "~하기 위해서는 / ~에 대하여"

- "분석을 시작하기 위해서는 파일이 필요합니다" → **"분석하려면 파일을 올려주세요"**
- "주문에 대하여 안내해 드리겠습니다" → **"주문 안내를 드릴게요"**

## 9. 과한 격식·공손 — "~해 주시기 바랍니다"

격식이 필요해도 UI에선 늘어진다.
- "다시 시도해 주시기 바랍니다" → **"다시 시도해 주세요"**
- "잠시만 기다려 주시기 바랍니다" → **"잠시만요"** / "잠시만 기다려 주세요"

## 10. AI·기계 번역 특유의 hype·과설명

- "단 한 번의 클릭으로 모든 것을" → 과장. UI에선 사실만: **"한 번에 정리해 드려요"**
- "혁신적인 / 강력한 / 압도적인" 수식어 → UI 카피에서 거의 항상 군더더기. 뺀다.
- 기능을 길게 설명("본 기능은 ~를 활용하여 ~합니다") → 사용자 이득 한 줄로.

---

## 빠른 체크 (문구 하나 통과시킬 때)
1. "성공적으로", "~하실 수 있습니다", "오류가 발생", "되어집니다" 들어갔나? → 십중팔구 번역투.
2. 사용자를 탓하나? → 행동 안내로.
3. 영어 원문을 그대로 옮긴 명사 나열인가? → 동사로 풀기.
4. 빼도 뜻이 사는 단어 있나? → 뺀다.
5. 사람이 옆에서 이렇게 말하나? → 소리 내 읽어보기.
